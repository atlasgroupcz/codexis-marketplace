# CODEXIS Marketplace

Tento repozitar obsahuje distribucni pluginy pro CODEXIS AI 2.0 marketplace.
Pluginy mohou rozsirovat platformu o skilly, agenty, MCP konektory, hooky a
UI komponenty.

## Dokumentace

- Dokumentace: https://docs.codexis.cz/
- CODEXIS AI 2.0 overview: https://docs.codexis.cz/codexis-ai-2-0/
- Plugin system: https://docs.codexis.cz/codexis-ai-2-0/capabilities/plugin-system
- Rychly start: https://docs.codexis.cz/guide/rychly-start

## Poznamka

Tento repozitar je distribucni vystup marketplace balicku generovany z
primarniho zdrojoveho repozitare.
