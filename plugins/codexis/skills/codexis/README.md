# CODEXIS — Vyhledávání v právní databázi

## Co to umí?

Tento skill umožňuje vyhledávat a číst dokumenty z právní databáze CODEXIS přímo v konverzaci. Stačí popsat, co hledáte, a skill zajistí vyhledání, načtení textu a zobrazení výsledků s přímými odkazy do databáze.

## Jaké zdroje jsou k dispozici?

| Zdroj | Popis |
|-------|-------|
| **Česká legislativa** | Zákony, vyhlášky, nařízení vlády, obecní vyhlášky — v aktuálním i historickém znění |
| **Slovenská legislativa** | Slovenské právní předpisy |
| **Česká judikatura** | Rozhodnutí Nejvyššího soudu, Ústavního soudu, krajských soudů a dalších |
| **Legislativa EU** | Nařízení a směrnice Evropské unie |
| **Judikatura EU** | Rozhodnutí Soudního dvora EU |
| **Právní literatura** | Odborné publikace a články |
| **Komentáře** | Komentáře LIBERIS k jednotlivým zákonům |
| **Vzory smluv** | Šablony a vzory pro běžné smluvní situace |

## Příklady použití

- *„Najdi mi aktuální znění § 89 občanského zákoníku."*
- *„Jaká je judikatura Ústavního soudu k náhradě škody?"*
- *„Ukaž mi směrnice EU o ochraně osobních údajů."*
- *„Co říká zákoník práce o výpovědi ze strany zaměstnavatele?"*
- *„Najdi vzor kupní smlouvy na nemovitost."*

## Co dostanete ve výsledku?

- Přesný text právního předpisu nebo rozhodnutí
- Přímé odkazy do databáze CODEXIS (kliknutím se otevře plný dokument)
- Možnost procházet strukturu dokumentu po jednotlivých paragrafech či článcích
- Související judikaturu a dokumenty k danému předpisu
